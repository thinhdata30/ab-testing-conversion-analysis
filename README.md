# A/B Testing Conversion Analysis

## Overview
This project analyzes the conversion rate difference between an advertisement group and a PSA group using A/B testing.

## Tools
- Python
- Pandas
- Seaborn
- Statsmodels

## Results
- Ad conversion rate: 2.55%
- PSA conversion rate: 1.78%
- p-value < 0.05

## Conclusion
Ads significantly improve conversion rate compared to PSA.
